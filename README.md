# Python图书借阅管理系统简介

## 核心功能
- 登录注册
- 浏览图书
- 图书管理
- 分类管理
- 用户管理
- 借阅管理
- 个人信息修改

## 适用场景
- 学校图书馆
- 企业图书角
- 公共图书馆
- 个人图书管理

## 技术特点
- 使用 Django 框架构建
- 基于 MySQL 数据库存储
- 使用 HTML、CSS 和 JavaScript 进行前端开发
